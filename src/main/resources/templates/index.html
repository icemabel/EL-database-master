Each study contains the following information:

  • Study Code: Unique identifier (e.g., CMQ_STD_001)
  • Document Codes: Protocol and report references
  • Material Type: Type of material being studied
  • Study Level: Classification level (A, B, C)
  • Risk Level: Safety risk assessment (Low, Medium, High)
  • Number of Samples: Sample count and identifiers
  • Object of Study: What is being analyzed
  • Responsible Person: Study coordinator
  • Status: Current study status
